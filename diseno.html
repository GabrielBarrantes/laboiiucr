Selector nivel de offset:

Para la selección del nivel DC del circuito se utilizó un amplificador sumador de offset el cual mediante la manipulación del potenciómetro P1 se elige si elevar o disminuir el nivel DC, incluso para valores negativos de tensión. Como medida de protección se decide utilizar un nivel DC máximo de 7.211 V. En las figuras se puede observar el diagrama del circuito propuesto para esta configuración y los niveles de tensión cuando la posición del potenciómetro está en 0%, 50% y 100%.

Selector de salida:

Para la selección del tipo de señal que el usuario desee obtener en la salida del generador se utilizará un conmutador de un polo y dos tiros para cada señal de forma que solo una de estas puede estar habilitada en conducción. Esta señal será acoplada a un amplificador sumador de ganancia unitaria al cual se le agrega la el nivel DC previamente mencionado. En la figura se observa el diagrama del circuito a implementar con los tres interruptores, donde siempre el tiro que no se usa está puesto a tierra. Además se toma la decisión de que para tener la condición de ganancia unitaria en el cual R2=3R1 se usa tres resistores de 10 kΩ nominal en serie.
